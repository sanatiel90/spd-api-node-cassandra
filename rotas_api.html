http://localhost:3000/api/login Loga usuário no sistema login: o username do usuário
password: senha do usuário 		{ "status": "200", "mensagem": "autenticado com sucesso!", "usuario": "rodrigo", "token": "eyJhbGciOlMkZCnxCUmMh8HOPDVYk" }
OBS: O token de acesso deve então ser armazenado pela aplicação que está consumindo a API e enviado em cada nova requisição que for feita à API. O armazenamento pode ser feito por exemplo usando Cookies, LocalStorage ou Sessão. O envio do token pode ser feito no BODY da requisição ou na querystring da URL. 		{ "status": "401", "mensagem": "credenciais inválidas" },
{ "status": "400", "mensagem": "falha ao autenticar token" },
{ "status": "403", "mensagem": "token de acesso não informado" }
http://localhost:3000/api/contacts Cadastra um novo contato para o usuário nome_contato: nome do contato
tel_contato: telefone do contato
token: o token de acesso gerado no login 		{ "status": "201", "mensagem": "contato criado com sucesso!", "usuario": "rodrigo", "contato": "joao", "telefone": "998984541" } { "status": "400", "mensagem": "falha ao autenticar token" },
{ "status": "400", "mensagem": "informe o nome do contato" }
http://localhost:3000/api/contacts Lista todos contatos do usuário token: o token de acesso gerado no login
Exemplo de uso: http://localhost:3000/api/contacts?token=eyJhbGciOlMkZCnxCUmMh8HOPDVYk 		{ "status": "200", "contatos": [ { "user_login": "rodrigo", "cont_name": "joao", "cont_tel": "998984541" }, { "user_login": "rodrigo", "cont_name": "maria", "cont_tel": "985411472" }, { "user_login": "rodrigo", "cont_name": "ramon", "cont_tel": "998410147" } ] } { "status": "400", "mensagem": "falha ao autenticar token" },
{ "status": "400", "mensagem": "não existem contatos para este usuário" },
{ "status": "400", "mensagem": "não foi possível encontrar os contatos" }
http://localhost:3000/api/contacts/nome_contato Retorna um contato com base no nome informado na URI nome_contato: O nome do contato desejado, informado na URI
token: o token de acesso gerado no login
Exemplo de uso: http://localhost:3000/api/contacts/maria?token=eyJhbGciOlMkZCnxCUmMh8HOPDVYk 		{ "status": "200", "nome_contato": "maria", "tel_contato": "985411472" } { "status": "400", "mensagem": "falha ao autenticar token" },
{ "status": "400", "mensagem": "contato não encontrado" },
{ "status": "400", "mensagem": "não foi possível encontrar o contato" }
http://localhost:3000/api/contacts/nome_contato Deleta o contato informado na URI nome_contato: O nome do contato a ser deletado
token: o token de acesso gerado no login
Exemplo de uso: http://localhost:3000/api/contacts/maria?token=eyJhbGciOlMkZCnxCUmMh8HOPDVYk 		{ "status": "200", "mensagem": "contato deletado com sucesso" } { "status": "400", "mensagem": "falha ao autenticar token" },
{ "status": "401", "mensagem": "contato não encontrado" },
